### Phase A2 — Route and codebase hardening

**Agent route modularized:**
- Split 1,202-line `app/api/agent/route.ts` into focused modules under `lib/agent/`:
  - `types.ts` — all tool input/output type definitions
  - `tools.ts` — 19 Anthropic tool schema definitions
  - `system-prompt.ts` — `buildSystemPrompt()` and property/reference types

- Prevents cross-property mutation when a user is a member of multiple properties
